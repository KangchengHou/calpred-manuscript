# admix-prs-uncertainty

To install the package,
```bash
pip install -e .
```

To use the package (for example),
```python
import admix_prs
?admix_prs.calibrate_prs
```
## File structures

- `admix_prs/`: source code
- `notebooks/`: example analysis.
- `experiments/`: code to replicate all main figures.
